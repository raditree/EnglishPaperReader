## 🚀 v1.0.0 (2026-02-07)

### ✨ 新功能

#### 📚 熟词词库管理

1. **批量导入熟词**
   - 支持从文本粘贴导入
   - 支持上传文本文件（.txt/.md/.csv）
   - 自动提取英文单词并去重
   - 导入后立即更新统计数据

2. **词库浏览功能**
   - 分页显示所有熟词（每页50个）
   - 实时搜索过滤
   - 显示每个单词的查询次数和添加时间
   - 支持重试机制，增强稳定性

3. **导入历史管理**
   - 记录每次导入的批次信息
   - 显示导入时间、来源、单词数量
   - 支持按批次查看

4. **撤销导入功能**
   - 可撤销错误的导入批次
   - 撤销后自动从熟词中移除
   - 自动更新统计数据
   - 撤销操作会还原单词的掌握状态

#### 📊 统计功能增强

- **启动时自动初始化**: 应用启动时自动刷新统计数据
- **导入后自动更新**: 导入熟词后立即更新总词汇量和已掌握词汇量
- **数据库连接保护**: 添加异常处理，防止连接泄漏

## 📦 v1.0.0 (2026-02-05)

### 🎉 初始版本

#### 📖 论文阅读功能
- 自动从 arXiv RSS 获取最新论文
- 支持多分类订阅（AI、CC、Math 等）
- 自动下载 PDF 到本地分类存储
- 基于 PDF.js 的 Web 端 PDF 阅读器
- 护眼模式（色温/亮度调节、夜间模式）

#### 🔤 划词翻译功能
- 基于 ECDICT 的本地英汉词典（50万+单词）
- 支持划词即译，快捷键翻译 (Ctrl+M / ⌘+M)
- 单词本功能，自动记录查询历史
- 支持模糊搜索和中文反向查询
- 批量翻译缓存，提升阅读体验

#### 📊 学习统计功能
- 实时记录阅读时长和单词查询
- 每日学习数据统计（查询次数、词汇量等）
- 单词掌握度追踪（0-5级）
- 基于艾宾浩斯遗忘曲线的复习建议
- 分类阅读统计（不同学科领域）
- 可视化图表（Chart.js）

#### 🔄 自动化功能
- 后台每6小时自动获取最新论文
- 自动去重，避免重复下载
- 生成每日论文精选 Markdown 报告
